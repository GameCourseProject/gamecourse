Introduction

On GameCouse, students obtain awards for their actions. Awards are the most basic unit of information and can be associated with game elements For example, a user completing the requirements for obtaining a certain badge receives an award of type 'badge'.

Game Elements, for instance, can be customized for each course. To do so, simply go into the Modules pages inside a course, and disable or enable the desired ones.

To get help on a more specific topic, select it from the sidebar!
