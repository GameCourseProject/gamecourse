Data Tooltips
Small pop-up boxes that appear when you hover your mouse pointer over a data point. Tooltips are used to provide additional information about the data or element, helping users understand and interpret the chart more effectively.
Make use of the points in the chart or write expressions if needed.
To access points use this.x and this.y, which will access each point coordinates in the data series.

Check the following example:
For data series Serie1, with X-categories: 'Jan', 'Feb' and values:

Serie1 = [10, 15]

We can format the tooltips for X-axis to be: "The X value for " + this.x + " corresponds to Y value " + this.y + " in this series".

Which would iterate through all data points in graph [ [10 , Jan] , [15 , Feb] , ... ] and translate to:
The X value for 10 corresponds to Y value Jan.
The X value for 15 corresponds to Y value Feb.
