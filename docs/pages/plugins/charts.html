Charts

Part - Chart

This Chart part allows to display different types of charts to the users. There are a 4 types of charts:

  • Line - draws a line chart
  • Bar - draws a bar chart
  • Star - draws a star plot
  • Progress - draws a progress bar

To create a new chart it is required to register a chart in using a Module in PHP.

Register a new chart

To register a new chart, get the chart Module, and then invoke the registerChart($id, $processFunction) method, which receives the new chart id and the process function that will return the values of the chart.

$chartsModule = $this->getParent()->getModule('charts');
$chartsModule->registerChart('xpWorld', function(&$chart, $params, $visitor) {
    //..
});
