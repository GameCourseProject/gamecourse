Creating a list of something is done with the Repeat this component for every item in field.


Every page must have a single Block, that will act as the root of the page. For this reason, you can't repeat that Block component. You can, however, add another Block, or any other type of component, inside it, and then repeat that inner component.


To do so, go into its Edit modal, and in the Repeat this component for every item in field insert an expression that returns a collection. An example would be:

{skills.getUsersWithSkill(1)}

which returns a collection of users with the skill of id 1.


Although the field has "Repetition" in its name, that doesn't mean that every item has to look exactly the same! Using the example above, we could list the names of the users of the collection. To access the item of each iteration, use %item. Then you could just write in the field "Text" of a Text component:

{%item.name}

to display the name of each user.


Keep in mind that if you have the Repetition defined in a component, and then want to access the item in other, deeper components, you should create an auxiliary variable to save the %item value. This also improves the readability of your code. So, in the component being repeated, you would create an auxiliary variable named currentUser, with {%item} on its expression, and use it instead.
