Although GameCourse comes with Components ready to be used style-wise, you can also customize them to your taste.


To do so, in the Pages editor, when you go into a Component's edit options, you have the Styling section available when you scroll down. In the field Style, you can use CSS. To apply more than one style remember to separate them by ";"!


GameCourse also uses both Tailwind and DaisyUI. In case you're not familiar with these, Tailwind is a framework that allows styling without the need for CSS, by simply using class names. DaisyUI extends Tailwind. As such, to use them in the editor, you can apply those classes in the Class field. As you would use them normally when coding, here you don't need to use any separator between several classes.


Tailwind has such a wide variety of class names that we can't describe them all here, and we highly suggest checking out its documentation! But here's the gist of it...

  • bg-* changes the color of the background, if you replace * with one of the colors' names available in DaisyUI
  • text-* similarly, changes the color of the text
  • font-bold is self-explanatory. There's also thin, light, extrabold, etc.
  • w-*, for example w-8, defines the width
  • h-* defines the height
  • p-* for padding, and m-* for margin
